The System Protocol for Channels and Tasks

API

  1. PUBLISHER
  2. SUBSCRIBER
  3. SPAWN
  4. SEND

pub [capacity]

pub creates new publisher CAS cursor for writers. Returns system-wide cursor Id.

o) p: pub[16]

sub [publisher]

sub created new subscriber CAS cursor based on given writer cursor. Return system-wide cursor Id.

o) s: sub[p]

spawn[core;program;cursors]

sub created new subscriber CAS cursor based on given writer cursor. Return system-wide cursor Id.

o) s: sub[p]

snd[writer;data]

Send particular data to a given writer cursor. Returns nil if OK.

o) snd[p;42]

rcv[reader]

Returns data from the given reader cursor. If none then yields to scheduler for another task.

o) rcv[s]
